# UI Direction Sketches

Three standalone mockups exploring distinct visual directions for the dashboard. Each is a self-contained HTML file with inline CSS and mock data — open directly in a browser:

```bash
open sketches/1-terminal.html
open sketches/2-glass.html
open sketches/3-brutalist.html
```

These are throwaway design references, not production code. Once a direction is picked, we build it for real in `packages/ui/`.

## The three

1. **Terminal-dense** — operator console, monospace, ASCII dividers, dense info per pixel. "I'd open this in tmux."
2. **Soft-glass dark** — modern observability dashboard, frosted cards, smooth gradients, animated charts. "Linear meets Grafana."
3. **Brutalist mono** — editorial / printed report. Massive numbers, hard rules, single accent color. "Pin this to the wall."
